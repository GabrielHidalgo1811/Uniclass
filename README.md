# Uniclass - Plataforma de Gestión Académica Universitaria

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## 📚 Descripción

Uniclass es una aplicación web completa diseñada para ayudar a estudiantes universitarios a organizar su vida académica. Ofrece un calendario semanal interactivo, gestión de pruebas y notas, temarios de estudio, y recordatorios personalizados.

## ✨ Características Principales

### 📅 Horario Interactivo
- Calendario semanal con vista tipo Google Calendar
- Bloques de clase continuos con información detallada
- Colores personalizables para cada ramo
- Visualización de profesor, sala y tipo de clase

### 📝 Gestión de Ramos
- Crear y gestionar asignaturas
- Pruebas predeterminadas (1, 2, 3) con opción de agregar más
- Cálculo automático de promedio ponderado
- Sistema de temarios por prueba con checkboxes

### ✅ Sidebar Inteligente
- Vista rápida de todos los temarios
- Checkboxes inline para marcar temas completados
- Organización jerárquica: Ramo → Prueba → Temas
- Sin salir del dashboard principal

### 🔔 Sistema de Eventos
- Eventos anclados (se repiten cada semana)
- Eventos únicos (solo para esa semana)
- Selección por día de la semana y hora

## 🛠️ Stack Tecnológico

- **Frontend**: React 18 + Tailwind CSS + Shadcn/UI
- **Backend**: Supabase (PostgreSQL + Auth)
- **Email**: Resend API
- **Dates**: date-fns

## 🚀 Instalación Rápida

1. Clonar repo y instalar dependencias
2. Configurar Supabase y variables de entorno
3. Ejecutar scripts SQL de la base de datos
4. `yarn start` en el frontend

Ver documentación completa en el README principal.
